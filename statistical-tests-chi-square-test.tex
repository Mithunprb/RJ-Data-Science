\section{Experiment}\label{experiment}}

\textbf{Aim of this project is to perform a chi-square test to check
whether two variables are independent or not.}

\hypertarget{data}{%
\subsection{Data}\label{data}}

The data comes in from the aircraft wildlife strikes data set and it
contains records of wildlife strikes against aircrafts from 1990 to
2015. (\href{https://github.com/Mithunprb/RJ-FDS}{link for the dataset})
We would be looking at indepedence testing for these two scenarios:

\begin{itemize}
\tightlist
\item
  visibility and flight impact
\item
  If there is a different impact caused by Seagulls and Mourning doves.
\end{itemize}
